Individual Sensitivity Analysis

Perturbs parameters individually, one at a time and by the same percentage.

This type of analysis is useful to find to which parameters a variable of the model is most sensitive to.

The number of simulations will be equal to the number of parameters to analyze plus one, the simulation corresponding to the standard run.

Known limitations

Only parameters and variables of type Real are recognized. Not even renamings of the sort of 'type MyType = Real' are supported.

Arrays of any type are not supported either.
